# 📚 Study Buddy App

A simple and interactive **Study Buddy Application** designed to help students organize their study sessions, track progress, and stay consistent with their learning goals.

---

## 🚀 Features
## 🚀 Features

- 🗓️ Create and manage study schedules  
- ⏱️ Pomodoro-style focus sessions  
- 📊 Track daily/weekly study progress  
- 🎯 Set personal study goals  
- 🔔 Reminders for study sessions  
- 🧠 Lightweight and distraction-free interface
